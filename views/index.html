URL SHORTENER MICROSERVICE

Please see the below instructions!

Example calls for google.com:

  • https://url-microservice-2.glitch.me/new/http://google.com
  • https://url-microservice-2.glitch.me/new/https://google.com

Which will return:

{"original_url": "http(s)://google.com", "shortened_url": "https://url-microservice-2.glitch.me/short/#"}

Whereas the shortened url:

https://url-microservice-2.glitch.me/short/#

Will redirect to the original:

http(s)://google.com
